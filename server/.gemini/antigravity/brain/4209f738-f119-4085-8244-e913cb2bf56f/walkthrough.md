
# Placement Intelligence Platform - Walkthrough

## Overview
This platform transforms campus placements with AI-driven intelligence for Students, Colleges, and Recruiters.

## Recent Features Implemented

### 1. Student Features
- **Skill Intelligence Engine**: Visualizes skill gaps against target roles.
- **Alumni Network**: Connects students with alumni in dream companies.
- **Interview Intelligence**: Repository of real interview experiences.

### 2. College Features
- **Curriculum Gap Analysis**: Identifies market skill trends vs student proficiency.
- **At-Risk Radar**: Proactively flags students struggling with placements.
- **Recruiter CRM**: Manages corporate relationships and tracking.

### 3. Recruiter Features
- **Smart Shortlisting**: AI-matched candidates based on skills and projects.
- **Offer Acceptance Probability**: Predicts joining likelihood to reduce ghosting.
- **Automated Scheduling**: Auto-generates interview slots for shortlisted candidates.

## How to Verify
1.  **Login** as different roles (Student, College, Recruiter).
2.  **Navigate** using the Sidebar to new modules.
3.  **Interact** with widgets (e.g., "Auto-Schedule" button in Recruiter view).

## Architecture
- **Frontend**: React, Vite, Tailwind CSS, Recharts.
- **Backend**: Node.js, Express, MongoDB.
- **Intelligence**: Mock AI models for probability and gap analysis.
